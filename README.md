# android-common-utils
this is a common utils of android (used to fast develop). contains QuickAdapter of ListView,RecyclerView.ExpandListView. and so on.

## How to use ? 
``` java
dependencies {
     compile 'org.heaven7.core:commonutil:1.7.2'
}
